### Overview
This repository contains an analytical exploration of how Information Technology (IT) serves as the core infrastructure for modern transportation systems. The task involved preparing and delivering extempore speeches on the digital evolution of three major sectors.

### Topics Covered:
1. **IT in Automobile:** Focuses on software-defined vehicles, Battery Management Systems (BMS), and IoT-driven EV infrastructure.
2. **IT in Metro Rail:** Explores Automated Train Control (ATC), smart ticketing, and real-time transit data management.
3. **IT in Avionics:** Examines high-reliability systems like Flight Management Systems (FMS), Fly-by-Wire technology, and predictive maintenance.

### Deliverables
* **Word Document:** A consolidated file containing the full transcripts for all three extempore topics.
* **Analysis:** Each speech identifies the specific technologies (IoT, AI, Big Data) that have transitioned these industries from mechanical to digital operations.
